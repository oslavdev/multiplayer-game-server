# Multiplayer game Server

WebGL sci-fi multiplayer game.

Client: https://github.com/oslavdev/multiplayer-shooter-client

## Features

- Select and customize character
- Private maps
- Choose weapons
- Audio, video and written chat
- Selection of maps

## Tech

- Node.js 
- Docker
- GraphQL
- PostgreSQL
- Ably (realtime communication)
- Prisma

## Deploy

### Build command


### Web

yarn run dev
